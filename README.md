# NT Fit: An AI Powered Fitness Web Application

NT Fit is an intelligent AI-powered fitness web application that analyzes body type, predicts fitness category using deep learning, and provides personalized diet plans, workout recommendations, and progress tracking.

## 🚀 Project Overview

NT Fit is designed to help users improve their health and fitness using Artificial Intelligence.
The system can:
* Detect body type using AI models
* Perform live camera body analysis
* Generate personalized workout plans
* Suggest diet recommendations
* Track user fitness progress
This application combines deep learning, computer vision, and web development into one complete smart fitness platform.

## 🎯 Features

### ✅ AI Body Type Detection
Classifies users into:
* Fit
* Average
* Overweight
### ✅ Live Camera Analysis

Real-time webcam-based prediction and analysis.

### ✅ Personalized Workout Plans

Custom workout routines based on detected body type.

### ✅ Diet Recommendation System

Smart meal suggestions tailored to user fitness goals.

### ✅ Progress Monitoring

Track improvements over time.

### ✅ Deep Learning Integration

Powered by TensorFlow, Keras, and ResNet50.

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/Nandu2704/NT-Fit-An-AI-Powered-Fitness-Web-Application.git
cd NT-Fit-An-AI-Powered-Fitness-Web-Application
```

### 2. Backend Setup

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

### 3. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
